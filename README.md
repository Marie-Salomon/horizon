# Event Horizon - Cosmological Research Dashboard

A modern, responsive Flask web application designed as a secure portal for cosmological research. This project features a high-tech "Bento Grid" UI, dark/light mode persistence, and secure user authentication with password hashing.

## üöÄ Features

- **Secure Authentication**: User sign-up and login system with passwords hashed using `Werkzeug.security`.
- **Modern UI/UX**: 
  - **Glassmorphism**: Frosted glass effects on cards and navigation.
  - **Dark/Light Mode**: User-controlled theme toggle with `localStorage` persistence.
  - **Responsive Design**: Fully functional on mobile and desktop using a Bento-style grid.
- **Scientific Journals**: Documented articles on the Big Bang and Black Holes with external scientific references (NASA, ESA, EHT).
- **Smooth Navigation**: Full-height Hero section with smooth scroll interactions.
